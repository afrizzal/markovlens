# Hero Screenshots

Two hero screenshots are required for the README.md `## Screenshots` section.

## Required Images

| File | Page | View to Capture |
|------|------|-----------------|
| `brand-share-fan.png` | Brand Share | Monte Carlo tab — fan chart with 10th/50th/90th percentile bands visible |
| `churn-sankey.png` | Churn | Overview tab — temporal Sankey state-flow diagram |

## How to Capture

1. Run the app locally: `uv run streamlit run app/Home.py`
2. Navigate to Brand Share → Monte Carlo tab → run a forecast → screenshot the fan chart → save as `brand-share-fan.png`
3. Navigate to Churn → Overview tab → screenshot the Sankey diagram → save as `churn-sankey.png`
4. Place both PNGs in this directory (`docs/assets/screenshots/`)

Or capture from the live Streamlit Cloud URL after deploy.
